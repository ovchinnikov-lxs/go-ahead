# go-ahead

#### 1. Устанавливаем зависимости
```sh
npm install
```


#### 2. Билдим
```sh
npm run build
```

#### 3. Запускаем
```sh
npm run preview
```


#### Режим разработки
```sh
npm run dev
```

#### Проверка/фиксы линтеров
```sh
npm run lint:fix
```

#### Запуск тестов
```sh
npm run test
```
