# go-ahead

#### 1. Устанавливаем зависимости
#### 1.1 Для установки зависимости заменить в `.npmrc` `AUTH_TOKEN` на токен из сообщения  
```sh
npm install
```


#### 2. Билдим
```sh
npm run build
```

#### 3. Запускаем
```sh
npm run preview
```

### Dev

#### Режим разработки
```sh
npm run dev
```

#### Проверка/фиксы линтеров
```sh
npm run lint:fix
```

#### Запуск unit тестов
```sh
npm run unit:test
```

#### Запуск e2e тестов
```sh
npm run e2e:test
```
